Image Search Abstraction Layer

Image Search

https://img-.herokuapp.com/imagesearch/[searchTerms]?offset=[offset]

Example

https://img-.herokuapp.com/imagesearch/cats?offset=2

The ?offset=[offset] is optional, it provides pagenation through the results, if ommited the first 10 results are returned.
